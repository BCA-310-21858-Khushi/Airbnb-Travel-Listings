# 🏡 WanderLust - Airbnb Travel & Listings Platform

## 📌 About the Project

WanderLust is a Full Stack Airbnb-inspired travel and accommodation booking platform developed as a Final Year BCA Project. The platform allows users to explore travel destinations, view property listings, book accommodations, manage their bookings, and save favourite listings. It also provides dedicated Host and Admin functionalities for efficient property and user management.

---

## ✨ Features

- User Registration & Login with OTP Verification
- Secure Authentication using Passport.js
- User Profile Management
- Become a Host Feature
- Add, Edit & Delete Listings
- Upload Multiple Images using Cloudinary
- Search & Filter Listings
- Wishlist Management
- Booking System
- Payment Module
- Booking Notifications
- Host Dashboard
- Admin Dashboard
- Monthly Revenue Analytics
- PDF Report Generation
- Responsive Design

## 🚀 Installation

Clone the repository

```bash
git clone https://github.com/BCA-310-21858-Khushi/Airbnb-Travel-Listings.git
```

Go to project folder

```bash
cd Airbnb-Travel-Listings
```

Install dependencies

```bash
npm install
```

Create a `.env` file and add your environment variables.

Start the project

```bash
npm start
```
